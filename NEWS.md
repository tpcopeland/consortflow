# consortflow 0.1.0

* Initial release.
* `cf_init()` to start an exclusion flowchart from a data frame.
* `cf_exclude()` to sequentially apply exclusion criteria with count tracking.
* `cf_draw()` to render the flowchart to the active graphics device.
* `cf_save()` to export flowcharts as PNG, PDF, or SVG.
* S3 methods: `print()`, `summary()`, `plot()`.
